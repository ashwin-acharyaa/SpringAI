# SpringAI
Spring Boot + Spring AI Demo

This is a minimal Spring Boot application that demonstrates how to integrate Spring AI with OpenAI to expose a simple chat API.

## Overview
The application exposes a single REST endpoint that accepts a message and returns a response generated by an AI model. It is intended as a starting point for experimenting with AI features in Java using the Spring ecosystem.

## Tech Stack
Java 25

Spring Boot 4.0.4

Spring AI (OpenAI starter)

Gradle

## Project Structure
src/main/java/com/example/ai

AiApplication.java     - Main application entry point

src/main/java/com/example/ai/controller

ChatController.java      - REST controller

src/main/resources

application.properties   - Configuration

Create an OpenAI API key from:
https://platform.openai.com/api-keys

Add it to application.properties:

spring.ai.openai.api-key=YOUR_OPENAI_API_KEY
spring.ai.openai.chat.options.model=gpt-4o-mini

The application will start on port 8080.

## API
GET /chat

Query parameter:

message – input text for the AI model

Example:

http://localhost:8080/chat?message=Explain%20Spring%20AI

Response:
A plain text response generated by the AI model.

## How It Works

The controller receives the request

ChatClient sends the prompt to the configured model

The model generates a response

The response is returned as a string

## Notes

You need an active OpenAI account with billing enabled

If you receive a 429 insufficient_quota error, check your usage and billing settings
